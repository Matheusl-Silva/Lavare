# UI Kit · Configurações

Tela de configurações com navegação lateral interna (vertical tabs). Seções: Perfil, Empresa, Usuários e permissões, Métodos de pagamento, Notificações.

## Arquivos
- `index.html`
- `AppShell.jsx` — sidebar com "Configurações" ativo (rodapé).
- `SettingsNav.jsx` — navegação lateral interna.
- `ProfileSection.jsx` — formulário de perfil.
- `BusinessSection.jsx` — dados da empresa.
- `UsersSection.jsx` — lista de usuários com permissões.
- `PaymentsSection.jsx` — métodos de pagamento aceitos.
- `NotificationsSection.jsx` — preferências de notificação (toggles).

## Estados cobertos
- Switching entre seções
- Toggles ligados/desligados
- Lista de usuários com badge de função
- Tabela de métodos de pagamento com switch
